# mywebgame

### Installation

```
cd mywebgame
npm install
```

### Run

```
npm run dev
```



# Build Steps

You don't need do these steps if you **only want to run this project**, just follow `README.md`

If you want to **build a new project** like this project, you should do these steps.

## Basis

### Initialize Vue

```
npm install -g vue-cli
vue init webpack mywebgame
cd mywebgame
npm run dev
```

### Install PIXI

```
npm install pixi.js
```

### Using PIXI

```
import * as PIXI from 'pixi.js'
```



### Install Pharse

```
npm install --save p2  
```

```
npm install --save phaser  
```



### Install Three.js

```
npm install three
```

### Install Physijs

```
 npm install --save physijs-webpack 
```
